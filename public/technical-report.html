Yönetici Özeti
Platformun teknik vizyonu ve temel metrikler

Heidelberg Materials Innovation Exchange (NIE), Heidelberg Materials ve iştiraklerinin (Hanson, Italcementi, Indocement, CBR, Heidelberg Materials North America, Heidelberg Materials Technology Center ve diğerleri) Almanya, Birleşik Krallık, ABD, Endonezya, İtalya, Belçika, Hindistan ve Mısır portallarında çalışanların fikir üretmesini, ekip ve kulüp kurmasını, kararlarını Karar Kurulu'na taşımasını ve inovasyon projelerini hayata geçirmesini sağlayan çok dilli kurumsal inovasyon platformudur. Sistem; fikir borsası, ekip ve kulüp yönetimi (kendi sohbet alanlarıyla), gerçek zamanlı mesajlaşma, AI destekli değerlendirme ve analitik modüllerinden oluşmaktadır.

Bu rapor, platformun üretim ortamına alınması için öngörülen enterprise-grade teknoloji yığınını, mimari kararları, güvenlik gereksinimlerini ve aylık operasyon maliyetlerini kapsamaktadır.

Frontend Teknoloji Yığını
UI framework, state yönetimi ve performans katmanı
Frontend, Server Components + Streaming mimarisiyle kurgulanmıştır. İlk yükleme 0.8 sn'nin altında tutulmakta; borsada gerçek zamanlı fiyat güncellemeleri WebSocket üzerinden sunulmaktadır.
Core Framework Next.js 15 v15.3.2 App Router, Server Components, Streaming SSR, ISR ile edge-optimized rendering.
UI Library React 19 v19.1.0 Concurrent rendering, use() hook, Server Actions ve Suspense ile modern bileşen mimarisi.
Language TypeScript 5.5 strict mode Tam tip güvenliği, zod ile runtime validation, tRPC ile end-to-end tipler.
Styling Tailwind CSS 4 v4.0 + CSS vars Lightning CSS compiler, design token sistemi, dark mode ve responsive grid.
Animation Framer Motion v11.x Layout animations, shared element transitions, spring physics, GPU-accelerated mikro-animasyonlar.
Components shadcn/ui + Radix headless Erişilebilir, unstyled bileşen primitifleri. WCAG 2.1 AA uyumlu.
State Zustand + Jotai v4.x / v2.x Global state için Zustand, atomic state için Jotai. Devtools ve persist middleware.
Data Fetching TanStack Query v5.x Stale-while-revalidate, optimistic updates, infinite scroll, background sync.
Charts Recharts + D3 v2.x / v7.x Borsa grafikleri, sinerji radar, analitik panelleri için SVG tabanlı rendering.
4
Backend & API Katmanı
Sunucu mimarisi, API tasarımı ve job queue sistemi
Runtime Bun 1.x v1.1.x Node.js'e kıyasla 3x daha hızlı cold start. Native TypeScript, built-in bundler ve test runner.
API Framework Hono.js v4.x Edge-native, ultra lightweight REST API. Middleware zinciri, validation (zod), JWT auth dahili.
GraphQL Apollo Server 4 Federation v2 Schema-first GraphQL, DataLoader batching, persisted queries, subscription (realtime).
RPC Layer tRPC v11 + Zod End-to-end type safety, istemci-sunucu arasında sözleşme. Next.js ile tam entegrasyon.
Job Queue BullMQ v5.x + Redis AI analiz, e-posta bildirimleri, borsa fiyat güncellemeleri için asenkron iş kuyruğu.
ORM Drizzle ORM v0.30+ SQL-first, sıfır overhead ORM. TypeScript şema tanımları, otomatik migration.
Mikro-servis hazırlığı: Monolith olarak başlayan backend, bounded context prensibiyle modüllere ayrılmıştır: Fikir Servisi, Borsa Servisi, Ekip Servisi, Mesajlaşma Servisi, AI Servisi. Ölçeğe göre ayrı konteyner olarak deploy edilebilir.

Geliştirme, test ve deployment akışı

Girişimcilik Ekosistemi & Genişletilmiş Özellikler
Sosyal tahmin, mentörlük, hisse paylaşım ve AI gündem modülleri
Sosyal Tahmin Başarı Tahmin & AI Sıralama Modeli Borsa projelerinin başarı olasılıklarını tahmin eden topluluk analiz aracı. Topluluk tahmin girdileri ve AI analiz skorunun harmanlanması ile çalışan dinamik başarı olasılığı algoritması.
Mentörlük Matchmaking & Mentör Dizini Holding içi uzmanların yetkinlik ve tecrübelerine göre listelendiği mentör dizini. Proje ekiplerinin ihtiyaç duyduğu konularda mentörlük talebi oluşturabilmesini sağlayan akıllı eşleştirme.
Ekip & Hisse Paylaşımlı Ortaklık & %10 Ödül Kuralı Projelerin açık hisse oranı (openEquity) takibi. Rol bazlı katılım formlarında deneyim ve talep edilen ortaklık payı beyanı. Proje hayata geçtiğinde girişimci ödülünün %10'unun yatırımcıları arasında paylaştırılması.
Otomasyon AI Gündem Analiz Motoru Platform içi trendleri, borsa hacmini ve tamamlanan projeleri izleyen NLP tabanlı yapay zeka analisti. Tek tıkla otomatik olarak analiz ve özet raporlarını kurumsal gündeme post eden otomasyon sistemi.
Ekipler (v2.5) Bağımsız Üst Seviye Sayfa Ekipler, Stüdyo hub'ından ayrılarak kendi navigasyon sekmesine taşındı. Proje ekipleri ve sosyal kulüpler tek sayfada, ayrı sekmelerde yönetilir.
Kulüpler (v2.5) Üyelik & Kulüp Profili Çalışanlar ilgi alanlarına göre kulüp kurabilir veya mevcut kulübe katılabilir. Her kulübün kategori, üye listesi ve kurucu bilgisi vardır.
Kulüp Sohbeti (v2.5) Gerçek Zamanlı Kulüp Chat Her kulübün kendine ait sohbet akışı vardır; sadece o kulübün üyeleri mesajlaşır. Mesaj gönderimi ücretsizdir, herhangi bir coin maliyeti yoktur.
Hisse Politikası (v2.5) %10 Maksimum Sahiplik Kuralı Bir kullanıcı tek bir projenin/hissenin toplam arzının en fazla %10'una sahip olabilir. Limit aşan alım emirleri sistem tarafından otomatik reddedilir.
Başlangıç Değerlemesi (v2.5) Standart 100 SA Açılış Fiyatı Borsaya listelenen her yeni proje/fikir, kaynağına (sihirbaz, şikayet kutusu, hızlı giriş) bakılmaksızın 100 SA sabit açılış değerlemesiyle başlar.
Karar Kurulu Coin Sistemi (v2.5) 5.000 SA Karşılığında Doğrudan Taşıma Proje sahibi, kendi fikrini/projesini veya önemli bir karar önerisini 5.000 SA ödeyerek doğrudan Karar Kurulu inceleme listesine taşıyabilir. Bakiye yetersizse işlem engellenir; kesinti yalnızca kullanıcı onayından sonra gerçekleşir.
